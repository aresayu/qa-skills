## 测试用例模板

| 字段 | 说明 |
|------|------|
| 用例编号 | 模块-接口序号-用例类型-序号 |
| 用例标题 | 概括"在什么输入/条件下，预期什么结果" |
| 接口路径 | API请求路径 |
| 请求方法 | HTTP方法 |
| 用例类型 | 正向/反向/边界/鉴权/幂等 |
| 优先级 | P0/P1/P2 |
| 前置条件 | 需要的账号、数据状态 |
| 请求参数 | 具体参数与取值 |
| 预期结果 | HTTP状态码、业务码、错误码/错误信息 |

## 用例类型说明

| 类型 | 说明 | 优先级 |
|------|------|--------|
| 正向测试 | 正常流程验证 | P0 |
| 反向测试 | 异常/错误验证 | P0-P1 |
| 边界值测试 | 边界条件验证 | P1 |
| 鉴权测试 | 安全相关验证 | P0-P1 |
| 幂等测试 | 重复操作验证 | P1-P2 |
| 分页测试 | 分页查询验证 | P1 |

## 覆盖维度说明

### 1. 请求参数校验
- 必填参数缺失
- 参数类型错误
- 参数格式错误
- 参数长度超限
- 参数值超出范围

### 2. 响应验证
- 成功响应结构和字段
- 错误码对应关系
- 错误信息准确性

### 3. 鉴权与权限
- 无token访问
- token无效/过期
- 越权访问

### 4. 业务逻辑
- 正常流程
- 异常流程
- 边界条件
- 状态流转

## 通用错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权/鉴权失败 |
| 403 | 无权限访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
