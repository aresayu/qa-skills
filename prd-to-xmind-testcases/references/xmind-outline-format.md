# XMind 导入格式与步骤

本文档说明如何将生成的测试用例Markdown文件导入XMind。

---

## 一、支持的格式

XMind支持以下格式导入：
- **Markdown (.md)** ✅ 推荐
- **OPML (.opml)**
- **MindManager (.mm)**
- **FreeMind (.mm)**
- **MindNode (.mindnode)**

**不支持**：纯文本(.txt)直接导入

---

## 二、Markdown格式规范

### 标题层级对应XMind层级

```markdown
# 一级标题 → XMind中心主题
## 二级标题 → XMind二级主题
### 三级标题 → XMind三级主题
#### 四级标题 → XMind四级主题
- 列表项 → XMind子节点
```

### 示例

```markdown
# 电商系统测试用例

## 用户模块
  ### 登录功能
    - 正常登录_输入正确账号密码
      - 步骤：输入正确账号密码，点击登录
      - 预期：登录成功

## 订单模块
  ### 下单功能
    - 正常下单_库存充足
```

导入XMind后对应结构：
```
🌗 电商系统测试用例
├── 👤 用户模块
│   └── 🔑 登录功能
│       ├── ✓ 正常登录_输入正确账号密码
│       └── ...
└── 📦 订单模块
    └── 🛒 下单功能
        ├── ✓ 正常下单_库存充足
        └── ...
```

## 三、导入步骤

### 方法一：XMind内置导入

1. 打开XMind软件
2. 点击菜单：`文件` → `导入` → `Markdown`
3. 选择生成的`.md`文件
4. 确认导入

### 方法二：拖拽导入

1. 打开XMind软件
2. 新建空白思维导图
3. 将`.md`文件拖拽到XMind窗口中
4. 选择导入方式

### 方法三：复制粘贴

1. 打开XMind软件
2. 新建空白思维导图
3. 选中中心主题
4. 复制Markdown中的内容（带层级）
5. 粘贴到XMind

---

## 四、注意事项

1. **层级清晰**：确保Markdown标题层级正确（# ## ### ####）
2. **列表缩进**：使用2个空格或Tab缩进表示子节点
3. **避免特殊字符**：节点名称避免使用XMind不支持的特殊字符
4. **图片处理**：Markdown中的图片链接不会被导入，需手动添加
5. **编码**：确保.md文件编码为UTF-8

---

## 五、快捷操作

- 导入后可在XMind中：
  - 调整节点位置
  - 添加图标/标记
  - 添加备注
  - 导出为其他格式（PDF、图片等）
