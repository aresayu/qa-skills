# 01 新增扣缴义务人

**接口路径**：`POST /api/v2/tax/withholding-agent/create`
**接口名称**：新增扣缴义务人

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| agentCode | 扣缴义务人编码 | String | 是 | 4-16位字母数字组合 |
| agentName | 扣缴义务人名称 | String | 是 | - |
| nsrsbh | 纳税人识别号 | String | 是 | 15/17/18/20位字母数字 |
| sbmm | 申报密码 | String | 是 | 8-20位 |
| passwordVerificationType | 密码校验类型 | String | 是 | - |
| areaid | 报税所属地区编码 | String | 是 | 6位数字 |
| djxhid | 主管税务机关ID | String | 是 | 通过[08 预获取税局任务ID](./08-预获取税局任务ID.md)获取taskId，再通过[09 根据任务ID查询税务机关](./09-根据任务ID查询税务机关.md)获取 |
| zgswjg | 主管税务机关名称 | String | 是 | 同上 |
| pkGroup | 所属集团 | String | 否 | - |
| pkTaxOrg | 所属组织 | String | 否 | - |
| bmbh | 部门编号 | String | 否 | - |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 返回数据 | Object | - |
| data.agentCode | 扣缴义务人编码 | String | - |
| data.agentName | 扣缴义务人名称 | String | - |
| data.nsrsbh | 纳税人识别号 | String | - |
| data.taxClassCode | 报税方案编码 | String | 是 | 关联 wa_tax_class.code |
| data.status | 账号状态 | String | - |

---

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "agentCode": "Agent001",
  "agentName": "XX科技有限公司",
  "nsrsbh": "91310000XXXXXXXX1K",
  "sbmm": "12345678",
  "passwordVerificationType": "01",
  "areaid": "310000",
  "djxhid": "DJXH20250301001",
  "zgswjg": "国家税务总局上海市税务局",
  "bmbh": "BM001"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "创建成功",
  "data": {
    "agentCode": "A001",
    "agentName": "测试扣缴义务人",
    "nsrsbh": "91310115MA1K4ABC12",
    "taxClassCode": "TC001",
    "status": "NORMAL"
  }
}
```
