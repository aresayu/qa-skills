# 05 搜索扣缴义务人

**接口路径**：`POST /api/v2/tax/withholding-agent/query`
**接口名称**：搜索扣缴义务人（支持搜索，多扣缴义务人选择场景）

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| agentCode | 扣缴义务人编码 | String | 否 | 模糊查询 |
| agentName | 扣缴义务人名称 | String | 否 | 模糊查询 |
| nsrsbh | 纳税人识别号 | String | 否 | 模糊查询 |
| zgswjg | 主管税务机关 | String | 否 | 模糊查询 |
| status | 账号状态 | String | 否 | 精确查询 |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 数据列表 | Array | 同列表返回结构 |

data 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| id | 扣缴义务人ID | String | - |
| agentCode | 扣缴义务人编码 | String | - |
| taxClassCode | 报税方案编码 | String | 关联 wa_tax_class.code |
| agentName | 扣缴义务人名称 | String | - |
| nsrsbh | 纳税人识别号 | String | - |
| zgswjg | 主管税务机关 | String | - |

---

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "agentName": "XX"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "success",
  "data": [
    {
      "id": "123",
      "agentCode": "Agent001",
      "taxClassCode": "TC001",
      "agentName": "XX科技有限公司",
      "nsrsbh": "91310000XXXXXXXX1K",
      "zgswjg": "国家税务总局上海市税务局"
    }
  ]
}
```
