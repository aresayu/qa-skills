# 07 获取报税地区列表

**接口路径**：`POST /api/v2/tax/withholding-agent/tax-regions`
**接口名称**：获取报税地区列表

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 地区列表 | Array | - |

data 数组元素字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| id | 地区ID | Long | - |
| code | 地区编码 | String | - |
| name | 地区名称 | String | - |

---

## 示例

**请求示例**

```json
{
  "cpId": 1001
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "success",
  "data": [
    {
      "id": 310000,
      "code": "310000",
      "name": "上海市"
    },
    {
      "id": 110000,
      "code": "110000",
      "name": "北京市"
    }
  ]
}
```
