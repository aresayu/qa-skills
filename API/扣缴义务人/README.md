# 扣缴义务人管理

**请求方式**：POST
**请求地址**：`/api/v2/tax/withholding-agent/{action}`
**接口描述**：扣缴义务人（Withholding Agent）是指个人所得税扣缴义务人，即企业税号登记信息。系统支持多扣缴义务人管理，每个扣缴义务人对应一个纳税人识别号（税号）。

> 接口安全（加密 + 签名）规则统一参考 [README · 接口安全](../README.md#4接口安全说明)。

## 场景说明

### 1. 企业首次使用个税通服务
企业首次使用个税通服务，需要先调用企业注册接口，校验录入的税号是否做过税务登记，税号正确才能进行人员报送等业务。

### 2. 主管税务机关选择
企业存在由于外验户或者多地经营等情况，存在多个登记序号，需要选择一个登记序号进行报税，可以调用主管税务机关接口进行选择。

### 3. 多登记序号/分部门管理
同一税号下不同的登记序号或者不同的分部门编号，都可以独立算税和独立申报个税。
