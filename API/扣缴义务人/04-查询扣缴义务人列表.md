# 04 查询扣缴义务人列表

**接口路径**：`POST /api/v2/tax/withholding-agent/list`
**接口名称**：查询扣缴义务人列表

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| agentCode | 扣缴义务人编码 | String | 否 | 模糊查询 |
| agentName | 扣缴义务人名称 | String | 否 | 模糊查询 |
| nsrsbh | 纳税人识别号 | String | 否 | 模糊查询 |
| zgswjg | 主管税务机关 | String | 否 | 模糊查询 |
| status | 账号状态 | String | 否 | 精确查询 |
| pageNum | 页码 | Integer | 否 | 默认 1 |
| pageSize | 每页条数 | Integer | 否 | 默认 10 |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data.list | 数据列表 | Array | - |
| data.total | 总条数 | Long | - |
| data.pageNum | 当前页 | Long | - |
| data.size | 每页条数 | Long | - |

data.list 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| id | 扣缴义务人ID | String | 用于兼容 AgentSelector 组件 |
| pkOrg | 扣缴义务人主键 | String | 与 id 一致 |
| agentCode | 扣缴义务人编码 | String | - |
| taxClassCode | 报税方案编码 | String | 关联 wa_tax_class.code |
| agentName | 扣缴义务人名称 | String | - |
| nsrsbh | 纳税人识别号 | String | - |
| zgswjg | 主管税务机关 | String | - |
| bmbh | 部门编号 | String | - |
| areaid | 报税所属地区 | String | - |
| status | 状态 | String | - |
| pkGroup | 所属集团 | String | - |
| pkTaxOrg | 所属组织 | String | - |
| cpId | 租户ID | Integer | - |
| createTime | 创建时间 | String | - |

---
