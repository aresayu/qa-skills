# 02 更新扣缴义务人

**接口路径**：`POST /api/v2/tax/withholding-agent/update`
**接口名称**：更新扣缴义务人

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| taxClassCode | 报税方案编码 | String | 是 | 通过[01 新增扣缴义务人](./01-新增扣缴义务人.md)响应获取 |
| agentName | 扣缴义务人名称 | String | 否 | - |
| sbmm | 申报密码 | String | 否 | 8-20位 |
| areaid | 报税所属地区编码 | String | 否 | 6位数字 |
| bmbh | 部门编号 | String | 否 | - |
| djxhid | 主管税务机关ID | String | 否 | - |
| zgswjg | 主管税务机关名称 | String | 否 | - |
| pkGroup | 所属集团 | String | 否 | - |
| pkTaxOrg | 所属组织 | String | 否 | - |

> **注意**：纳税人识别号（nsrsbh）编辑时不支持修改

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 返回数据 | Object | - |
| data.agentCode | 扣缴义务人编码 | String | - |
| data.agentName | 扣缴义务人名称 | String | - |
| data.nsrsbh | 纳税人识别号 | String | - |
| data.taxClassCode | 报税方案编码 | String | 关联 wa_tax_class.code |
| data.status | 账号状态 | String | - |

---

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "taxClassCode": "TC001",
  "agentName": "XX科技有限公司（更名）",
  "sbmm": "654321"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "更新成功",
  "data": {
    "agentCode": "A001",
    "agentName": "XX科技有限公司（更名）",
    "nsrsbh": "91310115MA1K4ABC12",
    "taxClassCode": "TC001",
    "status": "NORMAL"
  }
}
```
