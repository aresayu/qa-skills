# 03 获取扣缴义务人详情

**接口路径**：`POST /api/v2/tax/withholding-agent/detail`
**接口名称**：获取扣缴义务人详情

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| taxClassCode | 报税方案编码 | String | 是 | 通过[01 新增扣缴义务人](./01-新增扣缴义务人.md)响应获取 |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 详情数据 | Object | 见下方 |

data 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| agentCode | 扣缴义务人编码 | String | - |
| taxClassCode | 报税方案编码 | String | 关联 wa_tax_class.code |
| agentName | 扣缴义务人名称 | String | - |
| nsrsbh | 纳税人识别号 | String | - |
| password | 密码 | String | - |
| passwordVerificationType | 密码校验类型 | String | - |
| status | 状态 | String | PENDING_VERIFICATION/NORMAL/FAILED/LOCKED |
| pkGroup | 所属集团 | String | - |
| pkTaxOrg | 所属组织 | String | - |
| cpId | 租户ID | Integer | - |
| createTime | 创建时间 | String | - |
| filingPlans | 报税方案列表 | Array | - |

---

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "taxClassCode": "TC001"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "查询成功",
  "data": {
    "agentCode": "Agent001",
    "taxClassCode": "TC001",
    "agentName": "XX科技有限公司",
    "nsrsbh": "91310000XXXXXXXX1K",
    "status": "NORMAL",
    "cpId": 1001,
    "createTime": "2025-03-01T10:00:00"
  }
}
```
