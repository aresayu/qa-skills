# 09 根据任务ID查询税务机关

**接口路径**：`POST /api/v2/tax/withholding-agent/tax-authority-by-taskId`
**接口名称**：根据税局任务ID查询主管税务机关

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| taskId | 税局任务ID | String | 是 | 预获取税局任务ID接口返回 |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 返回数据 | Object | - |
| data.djxhid | 主管税务机关ID | String  | - |
| data.zgswjg | 主管税务机关名称 | String | - |

---

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "taskId": "TASK20250301001"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "success",
  "data": {
    "djxhid": "DJXH20250301001",
    "zgswjg": "国家税务总局上海市税务局"
  }
}
```
