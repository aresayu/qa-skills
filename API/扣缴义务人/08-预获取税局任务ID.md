# 08 预获取税局任务ID

**接口路径**：`POST /api/v2/tax/withholding-agent/tax-authority`
**接口名称**：预获取税局任务ID

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| nsrsbh | 纳税人识别号 | String | 是 | 税号 |
| areaid | 行政区划编码 | String | 是 | 6位数字 |
| bmbh | 部门编号 | String | 否 | - |
| sbmm | 申报密码 | String | 否 | - |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 返回数据 | Object | 包含 taxTaskId |

---

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "nsrsbh": "91310000XXXXXXXX1K",
  "areaid": "310000",
  "bmbh": "BM001"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "success",
  "data": {
    "taxTaskId": "TASK20250301001"
  }
}
```

---

## 说明

此接口调用税局 `hdxxxz` 接口，获取税局任务ID。创建扣缴义务人时，需先调用此接口获取 `taxTaskId`，再通过「根据任务ID查询税务机关」接口查询主管税务机关信息（djxhid、zgswjg），最后将主管税务机关信息传入「新增扣缴义务人」接口。
