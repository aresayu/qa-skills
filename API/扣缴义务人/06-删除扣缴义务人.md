# 06 删除扣缴义务人

**接口路径**：`POST /api/v2/tax/withholding-agent/delete`
**接口名称**：删除扣缴义务人（软删除）

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| taxClassCode | 报税方案编码 | String | 是 | 关联 wa_tax_class.code |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data | 返回数据 | Object | 包含受理ID等信息 |

data 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| status | 返回状态 | String | 成功，失败 |
| msg | 消息内容 | String | 失败原因 |


---

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "taxClassCode": "TC001"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "删除成功",
  "data": {
    "status": "成功",
    "msg": "删除成功"
  }
}
```
