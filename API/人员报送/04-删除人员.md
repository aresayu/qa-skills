# 04 删除人员

**接口路径**：`POST /api/v2/tax/person/delete`
**接口名称**：删除人员（软删除）

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | String | 是 | - |
| pkPerson | 人员PK | String | 是 | - |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |

---

## 示例

**请求示例**

```json
{
  "cpId": "1001",
  "pkPerson": "PSN20250301001"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "删除成功"
}
```
