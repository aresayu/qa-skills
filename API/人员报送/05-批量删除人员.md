# 05 批量删除人员

**接口路径**：`POST /api/v2/tax/person/batch-delete`
**接口名称**：批量删除人员（软删除）

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | String | 是 | - |
| pkPersons | 人员PK列表 | Array | 是 | 要删除的人员PK列表 |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data.deleted | 删除数量 | Integer | - |

---

## 示例

**请求示例**

```json
{
  "cpId": "1001",
  "pkPersons": ["PSN20250301001", "PSN20250301002", "PSN20250301003"]
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "批量删除成功",
  "data": {
    "deleted": 3
  }
}
```
