# 02 新增人员

**接口路径**：`POST /api/v2/tax/person/create`
**接口名称**：新增人员

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 企业CP_ID | String | 否 | - |
| bsny | 报送月份 | String | 是 | yyyy-mm-dd，不能大于当前系统时间 |
| psnType | 境内/境外人员 | String | 否 | 用来区分人员报送类型 |
| pkPerson | 人员PK | String | 否 | 修改时必填 |
| gh | 工号 | String | 否 | - |
| xm | 姓名 | String | 是 | - |
| zzlx | 证件类型 | String | 是 | 见[证照类型字典](../附录/证照类型字典.md) |
| zzhm | 证件号码 | String | 是 | - |
| kjsrybm | 扣缴义务人编码 | String | 是 | - |
| kjsry | 扣缴义务人 | String | 是 | - |
| gj | 国籍(地区) | String | 否 | 境外人员必填 |
| csny | 出生日期 | String | 否 | - |
| xb | 性别 | String | 否 | 男/女 |
| lxdh | 手机号码 | String | 否 | - |
| sfgy | 任职受雇从业类型 | String | 是 | - |
| qtqksm | 其他情况说明 | String | 否 | - |
| rzsgrq | 任职受雇从业日期 | String | 否 | - |
| lzrq | 离职日期 | String | 否 | - |
| sfbfgz | 是否离职后补发工资 | String | 否 | 0-否 / 1-是 |
| bfyf | 补发月份 | String | 否 | YYYYMM |
| sdyf | 所得月份 | String | 是 | YYYYMM |
| sfkcjcfy | 是否扣减减除费用 | String | 否 | 0-否 / 1-是 |
| bz | 备注 | String | 否 | - |
| zw | 职务 | String | 否 | - |
| lxdz | 联系地址 | String | 否 | - |
| taxClassCode | 报税方案编码 | String | 是 | - |
| xl | 学历 | String | 否 | - |
| sfgjzj | 是否高级专家 | String | 否 | - |
| rzndjyqk | 任职年度就业情况 | String | 否 | - |
| yggh | 员工工号 | String | 否 | - |
| khyh | 开户银行 | String | 否 | - |
| khyhSheng | 开户银行省份 | String | 否 | - |
| yhzh | 银行账号 | String | 否 | - |
| lxdzSheng | 经常居住地（省） | String | 否 | - |
| lxdzShi | 经常居住地（市） | String | 否 | - |
| lxdzQx | 经常居住地（区县） | String | 否 | - |
| lxdzJd | 经常居住地（街道） | String | 否 | - |
| lxdzXxdz | 经常居住地（详细地址） | String | 否 | - |
| hjszdSheng | 户籍所在地（省） | String | 否 | - |
| hjszdShi | 户籍所在地（市） | String | 否 | - |
| hjszdQx | 户籍所在地（区县） | String | 否 | - |
| hjszdJd | 户籍所在地（街道） | String | 否 | - |
| hjszdXxdz | 户籍所在地（详细地址） | String | 否 | - |
| sfcj | 是否残疾 | String | 否 | 0-否 / 1-是 |
| cjzjlx | 残疾证件类型 | String | 否 | - |
| cjzh | 残疾证号 | String | 否 | - |
| sfgl | 是否孤老 | String | 否 | 0-否 / 1-是 |
| sfls | 是否烈属 | String | 否 | 0-否 / 1-是 |
| lszh | 烈属证号 | String | 否 | - |
| grtzze | 个人投资总额 | BigDecimal | 否 | - |
| grtzbl | 个人投资比例 | BigDecimal | 否 | - |
| scrjsj | 首次入境时间 | String | 否 | - |
| yjljsj | 预计离境时间 | String | 否 | - |
| qtzzlx | 其他证照类型 | String | 否 | - |
| qtzzzm | 其他证照号码 | String | 否 | - |
| csd | 出生国家(地区) | String | 否 | - |
| sssy | 涉税事由 | String | 否 | - |
| xmzw | 中文名 | String | 否 | - |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data.pkPerson | 人员PK | String | 新增成功返回 |

---

## 示例

**请求示例**

```json
{
  "cpId": "1001",
  "xm": "张三",
  "zzlx": "A",
  "zzhm": "310101199001011234",
  "kjsrybm": "Agent001",
  "kjsry": "XX科技有限公司",
  "sfgy": "01",
  "sdyf": "202503",
  "taxClassCode": "PLAN_abc123def456",
  "lxdh": "13800138000"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "创建成功",
  "data": {
    "pkPerson": "PSN20250301001"
  }
}
```
