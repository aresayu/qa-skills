# 03 修改人员

**接口路径**：`POST /api/v2/tax/person/update`
**接口名称**：修改人员

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 企业CP_ID | String | 是 | - |
| pkPerson | 人员PK | String | 是 | 修改时必填 |
| gh | 工号 | String | 否 | - |
| xm | 姓名 | String | 否 | - |
| zzlx | 证件类型 | String | 否 | - |
| zzhm | 证件号码 | String | 否 | - |
| kjsrybm | 扣缴义务人编码 | String | 否 | - |
| kjsry | 扣缴义务人 | String | 否 | - |
| csny | 出生日期 | String | 否 | - |
| xb | 性别 | String | 否 | - |
| lxdh | 手机号码 | String | 否 | - |
| sfgy | 任职受雇从业类型 | String | 否 | - |
| qtqksm | 其他情况说明 | String | 否 | - |
| rzsgrq | 任职受雇从业日期 | String | 否 | - |
| lzrq | 离职日期 | String | 否 | - |
| sfbfgz | 是否离职后补发工资 | String | 否 | - |
| bfyf | 补发月份 | String | 否 | - |
| sdyf | 所得月份 | String | 否 | - |
| sfkcjcfy | 是否扣减减除费用 | String | 否 | - |
| bz | 备注 | String | 否 | - |
| zw | 职务 | String | 否 | - |
| lxdz | 联系地址 | String | 否 | - |
| taxClassCode | 报税方案编码 | String | 否 | - |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |

---

## 示例

**请求示例**

```json
{
  "cpId": "1001",
  "pkPerson": "PSN20250301001",
  "xm": "张三（修改）",
  "lxdh": "13900139000"
}
```

**响应示例**

```json
{
  "success": true,
  "msg": "更新成功"
}
```
