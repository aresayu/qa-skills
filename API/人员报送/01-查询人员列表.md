# 01 查询人员列表

**接口路径**：`POST /api/v2/tax/person/query`
**接口名称**：查询人员列表

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | String | 是 | - |
| kjsrybm | 扣缴义务人编码 | String | 否 | - |
| gh | 工号 | String | 否 | 模糊匹配 |
| xm | 姓名 | String | 否 | 模糊匹配 |
| zzhm | 证件号码 | String | 否 | 模糊匹配 |
| sfgy | 任职受雇从业类型 | String | 否 | - |
| bszt | 报送状态 | String | 否 | - |
| taxClassCode | 报税方案编码 | String | 否 | - |
| sdyf | 所得月份 | String | 否 | YYYYMM格式 |
| psnType | 人员类型 | String | 否 | DOMESTIC-境内 / FOREIGN-境外 |
| pageNum | 页码 | Integer | 否 | 默认 1 |
| pageSize | 每页条数 | Integer | 否 | 默认 20 |

---

## 响应参数

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| success | 是否成功 | Boolean | - |
| msg | 提示信息 | String | - |
| data.list | 数据列表 | Array | - |
| data.total | 总条数 | Long | - |
| data.pageNum | 当前页 | Long | - |
| data.size | 每页条数 | Long | - |

data.list 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| id | 主键ID | Long | - |
| pkPerson | 人员PK | String | - |
| gh | 工号 | String | - |
| xm | 姓名 | String | - |
| zzlx | 证件类型 | String | - |
| zzhm | 证件号码 | String | - |
| kjsrybm | 扣缴义务人编码 | String | - |
| kjsry | 扣缴义务人 | String | - |
| csny | 出生日期 | String | - |
| xb | 性别 | String | 男/女 |
| lxdh | 手机号码 | String | - |
| sfgy | 任职受雇从业类型 | String | - |
| bszt | 报送状态 | String | - |
| sdyf | 所得月份 | String | YYYYMM |
| taxClassCode | 报税方案编码 | String | - |
| xl | 学历 | String | - |
| sfgjzj | 是否高级专家 | String | - |
| rzndjyqk | 任职年度就业情况 | String | - |
| yggh | 员工工号 | String | - |
| sbyy | 失败原因 | String | - |
| sfbd | 是否并档 | String | 0-否 / 1-是 |
| bdjg | 并档结果 | String | - |
| bdsj | 并档时间 | String | - |
| sfcj | 是否残疾 | String | 0-否 / 1-是 |
| cjzjlx | 残疾证件类型 | String | - |
| cjzh | 残疾证号 | String | - |
| sfgl | 是否孤老 | String | 0-否 / 1-是 |
| sfls | 是否烈属 | String | 0-否 / 1-是 |
| lszh | 烈属证号 | String | - |
| gj | 国籍(地区) | String | - |
| createTime | 创建时间 | String | - |

---
