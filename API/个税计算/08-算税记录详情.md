# 算税记录详情

**请求方式**：POST  
**请求地址**：`/api/v2/tax/taxCompute/result/detail`  
**接口描述**：查询单条算税记录的完整税额明细，包含累计应纳税所得额、税率、速算扣除数、累计应扣缴税额等税局回填的完整计算字段。

> 接口安全（加密 + 签名）规则统一参考 [README · 接口安全](../README.md#4接口安全说明)。

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| id | 算税记录主键ID | Long | 是 | 来源于算税结果查询接口返回的 `id` 字段 |

---

## 响应参数

解密后 data 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| id | 算税记录ID | Long | - |
| xm | 姓名 | String | - |
| zzlx | 证照类型 | String | 字典编码，如 `A`=居民身份证 |
| zzhm | 证照号码 | String | 脱敏展示 |
| sdxm | 所得项目代码 | String | 税局编码，如 `0101`=正常工资薪金 |
| pch | 批次号 | Integer | - |
| sre | 本期收入 | BigDecimal | - |
| ybtse | 应补(退)税额 | BigDecimal | 正数为应补税，负数为应退税 |
| ljynssde | 累计应纳税所得额 | BigDecimal | - |
| sl | 税率 | BigDecimal | - |
| sskcs | 速算扣除数 | BigDecimal | - |
| ljynse | 累计应纳税额 | BigDecimal | - |
| ljjmse | 累计减免税额 | BigDecimal | - |
| ljyingkjse | 累计应扣缴税额 | BigDecimal | - |
| ykjse | 累计已预缴税额 | BigDecimal | - |
| status | 算税状态 | Integer | `-9`=处理失败 / `-1`=未处理 / `0`=正确 / `1`=有提示信息 |
| errorinfo | 错误信息 | String | 失败时税局返回的错误提示 |
| sourceType | 来源类型 | String | `SKJS`=个税计算 / `GYSS`=隔月试算 / `IMPORT`=直接导入 |
| isMultiPay | 多次发薪合并 | Integer | `0`=否 / `1`=是（合并多批次后按单条返回） |
| computeTime | 计算完成时间 | String | ISO 8601 格式 |

---
