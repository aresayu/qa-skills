# 薪酬数据导入

**请求方式**：POST  
**请求地址**：`/api/v2/tax/salaryData/import`  
**接口描述**：将当月薪酬数据写入算税记录表，作为后续触发算税的数据来源。上月所得（sysd）由系统自动从数据库查询，无需调用方传入。

> 接口安全（加密 + 签名）规则统一参考 [README · 接口安全](../README.md#4接口安全说明)。

---

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| nsrsbh | 扣缴义务人纳税人识别号 | String | 是 | - |
| taxClassCode | 报税方案编码 | String | 是 | 关联 wa_tax_class.code |
| sdyf | 所得月份 | String | 是 | 格式：YYYYMM |
| dysd | 当月所得记录列表 | Array | 是 | 最多 500 条，见附录 SalaryDataRecordDTO |

## 响应参数

外层统一封装：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| code | 状态码 | String | 200=成功 |
| msg | 消息 | String | - |
| data | 结果数据 | Object | 见下方 |

data 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| successCount | 成功导入数量 | Integer | - |
| failCount | 导入失败数量 | Integer | - |
| failDetails | 失败明细列表 | Array | 仅在有失败记录时返回 |
| failDetails[].zzhm | 失败的证照号码 | String | - |
| failDetails[].reason | 失败原因 | String | - |

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "nsrsbh": "91310000XXXXXXXX1K",
  "taxClassCode": "TC2025001",
  "sdyf": "202503",
  "dysd": [
    {
      "xm": "张三",
      "zzlx": "A",
      "zzhm": "310101199001011234",
      "itemCode": "01010101",
      "pch": 1,
      "sre": 15000.00,
      "jbylaobxf": 960.00,
      "jbylbxf": 240.00,
      "sybxf": 60.00,
      "zfgjj": 1200.00
    }
  ]
}
```

**响应示例**

```json
{
  "code": "200",
  "msg": "success",
  "data": {
    "successCount": 1,
    "failCount": 0,
    "failDetails": []
  }
}
```
