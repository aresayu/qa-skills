# 薪酬数据查询

**请求方式**：POST  
**请求地址**：`/api/v2/tax/salaryData/list`  
**接口描述**：分页查询指定报税方案、所得月份下的薪酬数据列表，支持按所得项目代码和算税状态过滤。

> 接口安全（加密 + 签名）规则统一参考 [README · 接口安全](../README.md#4接口安全说明)。

## 请求参数

| 字段名称 | 字段含义 | 字段类型 | 必填 | 说明 |
| :--- | :--- | :--- | :---: | :--- |
| cpId | 租户ID | Integer | 是 | - |
| taxClassCode | 报税方案编码 | String | 是 | 关联 wa_tax_class.code |
| sdyf | 所得月份 | String | 是 | 格式：YYYYMM |
| sdxm | 所得项目代码 | String | 否 | 如 0101，不传则查全部 |
| status | 算税状态 | Integer | 否 | -9=处理失败 / -1=未处理 / 0=正确 / 1=有提示信息 |
| pageNum | 页码 | Integer | 否 | 默认 1 |
| pageSize | 每页大小 | Integer | 否 | 默认 20，最大 500 |

---

## 响应参数

外层统一封装：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| code | 状态码 | String | 200=成功 |
| msg | 消息 | String | - |
| data | 结果数据 | Object | 见下方 |

data 字段：

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| total | 总记录数 | Long | - |
| list | 数据列表 | Array | 每条记录字段见下方 |

### list 单条记录字段

| 字段名称 | 字段含义 | 字段类型 | 说明 |
| :--- | :--- | :--- | :--- |
| id | 记录ID | Long | - |
| xm | 姓名 | String | - |
| zzlx | 证照类型 | String | 字典编码，如 "A" 代表居民身份证 |
| zzhm | 证照号码 | String | 脏敏展示 |
| sdxm | 所得项目代码 | String | 税局编码，如 0101=正常工资薪金 |
| pch | 批次号 | Integer | 同人同月多次发薪时的批次编号 |
| sre | 本期收入 | BigDecimal | - |
| ybtse | 应补(退)税额 | BigDecimal | 正数为应补税，负数为应退税 |
| status | 算税状态 | Integer | -9=处理失败 / -1=未处理 / 0=正确 / 1=有提示信息 |
| errorinfo | 错误信息 | String | 税局返回的错误提示（失败时） |
| sourceType | 来源类型 | String | SKJS=正常算税 / GYSS=隔月试算 / IMPORT=直接导入 |
| computeTime | 计算完成时间 | String | ISO 8601 格式 |

## 示例

**请求示例**

```json
{
  "cpId": 1001,
  "taxClassCode": "TC2025001",
  "sdyf": "202503",
  "status": 0,
  "pageNum": 1,
  "pageSize": 20
}
```

**响应示例**

```json
{
  "code": "200",
  "msg": "success",
  "data": {
    "total": 100,
    "list": [
      {
        "id": 1001,
        "xm": "张三",
        "zzlx": "A",
        "zzhm": "310***1234",
        "sdxm": "0101",
        "pch": 1,
        "sre": 15000.00,
        "ybtse": 1500.00,
        "status": 0,
        "errorinfo": null,
        "sourceType": "GYSS",
        "computeTime": "2025-03-20T10:30:00"
      }
    ]
  }
}
```
