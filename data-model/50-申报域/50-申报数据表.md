# 50-申报域 - 申报数据表

## 1. 申报汇总表（wa_tax_declaration）

### 表基本信息

| 属性 | 说明 |
|------|------|
| 表名 | wa_tax_declaration |
| 中文名 | 个税申报汇总表 |
| 说明 | 存储个税申报的汇总信息，按所得月份、纳税人识别号、所得类型聚合 |
| 关联 | 报税方案表(wa_tax_class)、人员申报明细表(wa_tax_declaration_detail) |

### 表结构

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT UNSIGNED | PK, AUTO_INCREMENT | 自增主键（内部使用） |
| declaration_biz_id | VARCHAR(64) | NOT NULL | 业务主键（雪花ID） |
| cp_id | INT | NOT NULL | 租户ID |
| tax_class_code | VARCHAR(32) | DEFAULT NULL | 报税方案编码（wa_tax_class.code） |
| nsrsbh | VARCHAR(20) | NOT NULL | 纳税人识别号 |
| nsrsbh_name | VARCHAR(100) | DEFAULT NULL | 扣缴义务人名称 |
| sdyf | VARCHAR(6) | NOT NULL | 所得月份（YYYYMM） |
| category_code | VARCHAR(10) | NOT NULL DEFAULT 'TOTAL' | 所得类型代码（wa_tax_income_category的code） |
| category_name | VARCHAR(50) | DEFAULT NULL | 所得类型名称 |
| current_income | DECIMAL(18,2) | DEFAULT 0.00 | 本期收入总额 |
| total_income | DECIMAL(18,2) | DEFAULT 0.00 | 累计收入累计 |
| tax_amount | DECIMAL(18,2) | DEFAULT 0.00 | 应纳税额总额 |
| tax_supplement | DECIMAL(18,2) | DEFAULT 0.00 | 应补(退)税额总额 |
| tax_payers | INT | DEFAULT 0 | 纳税人数（去重） |
| compute_status | INT | NOT NULL DEFAULT -1 | 计算状态 |
| declaration_type | VARCHAR(20) | NOT NULL DEFAULT 'NORMAL' | 申报类型 |
| declaration_status | VARCHAR(20) | NOT NULL DEFAULT 'UNREPORTED' | 申报状态 |
| error_msg | VARCHAR(500) | DEFAULT NULL | 失败错误信息 |
| dr | TINYINT(1) | NOT NULL DEFAULT 0 | 删除标记 |
| creator | VARCHAR(64) | DEFAULT NULL | 创建者 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| modifier | VARCHAR(64) | DEFAULT NULL | 修改者 |
| modify_time | DATETIME | NOT NULL | 修改时间 |

### 索引

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | id | PRIMARY | 主键索引 |
| uk_declaration_biz_id | declaration_biz_id | UNIQUE | 业务主键唯一索引 |
| uk_decl_logic | cp_id, nsrsbh, sdyf, category_code, dr | UNIQUE | 逻辑唯一约束 |
| idx_nsrsbh_sdyf | nsrsbh, sdyf | INDEX | 纳税人+月份索引 |

### 枚举值

| 枚举名称 | 字段 | 值 | 说明 |
|----------|------|-----|------|
| 申报类型 | declaration_type | NORMAL | 正常申报 |
| 申报类型 | declaration_type | CORRECTION | 更正申报 |
| 申报状态 | declaration_status | UNREPORTED | 未申报 |
| 申报状态 | declaration_status | REPORTED | 已申报 |
