# 45-扣除域 - 扣除数据表

## 1. 专项附加扣除表（wa_tax_deduction_special_detail）

### 表基本信息

| 属性 | 说明 |
|------|------|
| 表名 | wa_tax_deduction_special_detail |
| 中文名 | 专项附加扣除表（与个人养老金共用） |
| 说明 | 存储员工每月的专项附加扣除信息（子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人等） |
| 关联 | 报税方案表(wa_tax_class)、人员信息表(bd_psndoc) |

### 表结构

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| pk_wa_tax_deducti | VARCHAR(64) | PK | 主键 |
| cp_id | INT(11) | NOT NULL | 租户ID |
| pk_tax_class | VARCHAR(64) | NOT NULL | 报税方案主键（扣缴义务人） |
| pk_tax_org | VARCHAR(64) | NULL | 报税组织主键 |
| pk_group | VARCHAR(64) | NULL | 分组主键 |
| taxyear | VARCHAR(4) | NOT NULL | 报税年份 |
| taxperiod | VARCHAR(7) | NOT NULL | 报税月份（YYYY-MM） |
| pk_psndoc | VARCHAR(64) | NOT NULL | 人员表主键 |
| islocked | TINYINT(1) | NOT NULL DEFAULT 0 | 是否锁定（0-未锁定 1-已锁定） |
| sys_status | VARCHAR(20) | DEFAULT NULL | 同步状态（SUCCESS/FAILED/PENDING） |
| sys_result | VARCHAR(500) | DEFAULT NULL | 同步结果描述 |
| znjyzc | DECIMAL(10,2) | DEFAULT NULL | 累计子女教育 |
| yyezhfzc | DECIMAL(10,2) | DEFAULT NULL | 累计3岁以下婴幼儿照护费 |
| jxjyzc | DECIMAL(10,2) | DEFAULT NULL | 累计继续教育 |
| zfzjzc | DECIMAL(10,2) | DEFAULT NULL | 累计住房租金 |
| zfdklxzc | DECIMAL(10,2) | DEFAULT NULL | 累计住房贷款利息 |
| sylrzc | DECIMAL(10,2) | DEFAULT NULL | 累计赡养老人 |
| grylj | DECIMAL(10,2) | DEFAULT NULL | 累计个人养老金 |
| update_time | DATETIME | DEFAULT NULL | 更新时间 |

### 索引

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | pk_wa_tax_deducti | PRIMARY | 主键索引 |
| idx_cp_taxclass_period | cp_id, pk_tax_class, taxperiod | INDEX | 租户+方案+月份索引 |
| idx_psndoc_period | pk_psndoc, taxperiod | INDEX | 人员+月份索引 |
| idx_locked | islocked | INDEX | 锁定状态索引 |
| uk_psndoc_period | pk_psndoc, taxperiod, pk_tax_class | UNIQUE | 人员+月份+方案唯一索引 |

### 枚举值

| 枚举名称 | 字段 | 值 | 说明 |
|----------|------|-----|------|
| 同步状态 | sys_status | SUCCESS | 同步成功 |
| 同步状态 | sys_status | FAILED | 同步失败 |
| 同步状态 | sys_status | PENDING | 同步中 |

## 2. 6万减除费用确认表（wa_tax_60k_deduction_detail）

### 表基本信息

| 属性 | 说明 |
|------|------|
| 表名 | wa_tax_60k_deduction_detail |
| 中文名 | 6万减除费用人员确认表 |
| 说明 | 存储符合条件的员工6万减除费用确认信息，用于年度个税汇算清缴 |
| 关联 | 报税方案表(wa_tax_class)、人员信息表(bd_psndoc) |

### 表结构

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| pk_wa_tax_deducti | VARCHAR(64) | PK | 主键 |
| cp_id | VARCHAR(64) | NOT NULL | 租户ID |
| pk_tax_class | VARCHAR(64) | NOT NULL | 报税方案主键（扣缴义务人） |
| pk_tax_org | VARCHAR(64) | NULL | 报税组织主键 |
| pk_group | VARCHAR(64) | NULL | 分组主键 |
| taxyear | VARCHAR(4) | NOT NULL | 报税年份 |
| taxperiod | VARCHAR(7) | NULL | 报税月份（YYYY-MM） |
| pk_psndoc | VARCHAR(64) | NOT NULL | 人员表主键 |
| sfkc | VARCHAR(10) | DEFAULT NULL | 是否扣除减除费用（Y/N） |
| status | VARCHAR(20) | DEFAULT NULL | 状态（SUCCESS/FAILED/PENDING） |
| errInfo | VARCHAR(500) | DEFAULT NULL | 错误信息 |

### 索引

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | pk_wa_tax_deducti | PRIMARY | 主键索引 |
| idx_cp_taxclass_period | cp_id, pk_tax_class, taxperiod | INDEX | 租户+方案+月份索引 |
| idx_psndoc_period | pk_psndoc, taxperiod | INDEX | 人员+月份索引 |

### 枚举值

| 枚举名称 | 字段 | 值 | 说明 |
|----------|------|-----|------|
| 是否扣除 | sfkc | Y | 是 |
| 是否扣除 | sfkc | N | 否 |
| 状态 | status | SUCCESS | 成功 |
| 状态 | status | FAILED | 失败 |
| 状态 | status | PENDING | 处理中 |

## 3. 捐赠信息表（wa_tax_donation_detail）

### 表基本信息

| 属性 | 说明 |
|------|------|
| 表名 | wa_tax_donation_detail |
| 中文名 | 捐赠信息表 |
| 说明 | 存储员工的公益捐赠信息，用于税前扣除 |
| 关联 | 报税方案表(wa_tax_class)、人员信息表(bd_psndoc) |

### 表结构

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| pk_wa_tax_deducti | VARCHAR(64) | PK | 主键 |
| cp_id | INT(11) | NOT NULL | 租户ID |
| pk_tax_class | VARCHAR(64) | NOT NULL | 报税方案主键（扣缴义务人） |
| pk_tax_org | VARCHAR(64) | NULL | 报税组织主键 |
| pk_group | VARCHAR(64) | NULL | 分组主键 |
| taxyear | VARCHAR(4) | NOT NULL | 报税年份 |
| taxperiod | VARCHAR(7) | NOT NULL | 捐赠年月（YYYY-MM） |
| pk_psndoc | VARCHAR(64) | NOT NULL | 人员表主键 |
| donation_org_code | VARCHAR(18) | NOT NULL | 受赠单位统一社会信用代码 |
| donation_org_name | VARCHAR(200) | NOT NULL | 受赠单位名称 |

| deduction_ratio | DECIMAL(5,2) | NOT NULL | 扣除比例（0-100） |
| status | VARCHAR(20) | DEFAULT NULL | 状态（ACTIVE/INACTIVE） |
| update_time | DATETIME | DEFAULT NULL | 更新时间 |

### 索引

| 索引名 | 字段 | 类型 | 说明 |

| idx_org_code | donation_org_code | INDEX | 受赠单位税号索引 |
| uk_psndoc_cert_period | pk_psndoc, donation_cert_no, taxperiod, pk_tax_class | UNIQUE | 人员+凭证+月份+方案唯一索引 |

### 枚举值

| 枚举名称 | 字段 | 值 | 说明 |
|----------|------|-----|------|
| 状态 | status | ACTIVE | 有效 |
| 状态 | status | INACTIVE | 无效 |
