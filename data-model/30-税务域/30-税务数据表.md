# 30-税务域 - 税务数据表

## 1. 报税方案表（wa_tax_class）

### 表基本信息

| 属性 | 说明 |
|------|------|
| 表名 | wa_tax_class |
| 中文名 | 报税方案表（合并原wa_tax_setting字段） |
| 说明 | 存储企业税号登记信息和报税方案配置；原wa_tax_setting表的字段已合并到此表中 |
| 关联 | 报税区域表(wa_tax_region)、人员信息表(bd_psndoc) |

### 表结构

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 自增主键 |
| pk_tax_class | VARCHAR(32) | NOT NULL | 主键（PLAN_+RandomId） |
| pk_tax_setting | BIGINT | NULL | 关联wa_tax_setting.pk_tax_setting（保留用于过渡） |
| pk_group | VARCHAR(64) | NULL | 所属集团 |
| yxqq | VARCHAR(10) | NULL | 有效期起 |
| dr | TINYINT(1) | NOT NULL DEFAULT FALSE | 删除标记：0-正常/1-已删除 |
| pk_tax_org | VARCHAR(64) | NULL | 所属组织 |

**扣缴义务人字段（原wa_tax_setting）**：

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| agent_code | VARCHAR(64) | NULL | 扣缴义务人编码 |
| agent_name | VARCHAR(100) | NULL | 扣缴义务人名称/单位名称 |
| nsrsbh | VARCHAR(50) | NULL | 纳税人识别号 |
| sbmm | VARCHAR(255) | NULL | 申报密码（AES加密存储） |
| password_verification_type | VARCHAR(50) | NULL | 密码校验类型（默认PERSONAL_TAX） |
| status | VARCHAR(20) | NULL | 状态（PENDING_VERIFICATION-校验中/NORMAL-正常/FAILED-失败/LOCKED-已锁定） |

**报税方案字段**：

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| code | VARCHAR(32) | NULL | 报税方案编码 |
| areaid | VARCHAR(20) | NULL | 区域ID（外键关联wa_tax_region） |
| name | VARCHAR(100) | NULL | 报税方案名称 |
| bmbh | VARCHAR(32) | NULL | 部门编号 |
| djxhid | VARCHAR(64) | NULL | 主管税务机关等级序列号 |
| zgswjg | VARCHAR(200) | NULL | 主管税务机关名称 |
| cp_id | INT | NOT NULL | 租户ID |
| creator | VARCHAR(64) | NULL | 创建人 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| modifier | VARCHAR(64) | NULL | 修改人 |
| modify_time | DATETIME | NOT NULL | 更新时间 |

### 索引

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | id | PRIMARY | 主键索引 |
| uk_pk_tax_class | pk_tax_class | UNIQUE | 报税方案主键唯一索引 |
| idx_pk_tax_setting | pk_tax_setting | INDEX | 关联税号设置索引 |
| idx_code | code | INDEX | 方案编码索引 |
| idx_areaid | areaid | INDEX | 区域ID索引 |
| idx_cp_id | cp_id | INDEX | 租户ID索引 |
| idx_agent_code | agent_code | INDEX | 扣缴义务人编码索引 |
| idx_nsrsbh | nsrsbh | INDEX | 纳税人识别号索引 |
| idx_status | status | INDEX | 状态索引 |

### 外键约束

| 约束名 | 字段 | 参考表 | 参考字段 | 删除策略 |
|--------|------|--------|----------|----------|
| fk_tax_class_region | areaid | wa_tax_region | pk_tax_region | SET NULL |

| 中文名 | 算税任务队列表 |

| 字段 | 类型 | 约束 | 说明 |

| dr | TINYINT(1) | NOT NULL DEFAULT 0 | 删除标记（0-正常/1-已删除） |
| creator | VARCHAR(64) | NULL | 创建人 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| actionstarttime | VARCHAR(30) | NULL | 处理开始时间 |
| actionendtime | VARCHAR(30) | NULL | 处理结束时间 |
| actionno | VARCHAR(64) | NULL | 处理批次号（防并发重入） |

| 枚举名称 | 字段 | 值 | 说明 |

## 8. 所得项目表（wa_tax_income_item）

### 表基本信息

| 属性 | 说明 |
|------|------|
| 表名 | wa_tax_income_item |
| 中文名 | 所得底层项目字典表 |
| 说明 | 存储个税所得的底层项目（正常工资薪金、年终奖等） |
| 关联 | 所得子分类表(wa_tax_income_sub_category) |

### 表结构

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| sub_category_id | BIGINT | NOT NULL | 中层子分类ID（关联wa_tax_income_sub_category.id） |
| item_code | VARCHAR(20) | NOT NULL | 系统自有唯一编码（如01010101），全局唯一 |
| sdxm | VARCHAR(10) | DEFAULT NULL | 税局所得项目代码（如0101），无税局编码时为NULL |
| name | VARCHAR(100) | NOT NULL | 项目名称 |
| tax_method | VARCHAR(20) | NOT NULL DEFAULT 'CUMULATIVE' | 扣缴方式（CUMULATIVE=累计预扣法/FLAT_RATE=按次固定税率） |
| sort_order | INT | NOT NULL DEFAULT 0 | 排序 |
| enabled | TINYINT(1) | NOT NULL DEFAULT 1 | 是否启用（0-否/1-是） |
| dr | TINYINT(1) | NOT NULL DEFAULT 0 | 删除标记（0-正常/1-已删除） |
| remark | VARCHAR(200) | NULL | 备注 |
| creator | VARCHAR(64) | NULL | 创建人 |
| create_time | DATETIME | NOT NULL | 创建时间 |
| modifier | VARCHAR(64) | NULL | 修改人 |
| modify_time | DATETIME | NOT NULL | 更新时间 |

### 索引

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | id | PRIMARY | 主键索引 |
| uk_item_code | item_code | UNIQUE | 项目代码唯一索引 |
| idx_sub_category_id | sub_category_id | INDEX | 中层子分类ID索引 |
| idx_sdxm | sdxm | INDEX | 税局项目代码索引 |

### 枚举值

| 枚举名称 | 字段 | 值 | 说明 |
|----------|------|-----|------|
| 扣缴方式 | tax_method | CUMULATIVE | 累计预扣法 |
| 扣缴方式 | tax_method | FLAT_RATE | 按次固定税率 |

### 扣缴方式说明

| 方式 | 适用项目 | 说明 |
|------|----------|------|
| CUMULATIVE | 正常工资薪金、劳务报酬（累计）等 | 按累计预扣法计算 |
| FLAT_RATE | 年终奖、劳务报酬（不累计）等 | 按次或按月固定税率计算 |
