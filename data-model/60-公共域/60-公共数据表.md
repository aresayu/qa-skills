# 60-公共域 - 公共数据表

## 1. 操作日志表（operation_log）

### 表基本信息

| 属性 | 说明 |
|------|------|
| 表名 | operation_log |
| 中文名 | 操作日志表 |
| 说明 | 存储用户的所有操作日志，包含操作人、操作内容、变更前后数据快照等 |

### 表结构

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 操作日志ID |
| operation_time | DATETIME | NOT NULL | 操作时间 |
| operator_id | BIGINT | NOT NULL | 操作人用户ID |
| operator_name | VARCHAR(64) | NOT NULL | 操作人用户名 |
| operation_type | VARCHAR(32) | NOT NULL | 操作类型 |
| operation_desc | VARCHAR(255) | NOT NULL | 操作描述 |
| operation_result | VARCHAR(32) | NOT NULL | 操作结果 |
| target_type | VARCHAR(64) | DEFAULT NULL | 操作对象类型 |
| target_id | VARCHAR(128) | DEFAULT NULL | 操作对象标识 |
| target_name | VARCHAR(255) | DEFAULT NULL | 操作对象名称 |
| client_ip | VARCHAR(64) | DEFAULT NULL | 客户端IP地址 |
| before_snapshot | MEDIUMTEXT | DEFAULT NULL | 变更前数据快照(JSON) |
| after_snapshot | MEDIUMTEXT | DEFAULT NULL | 变更后数据快照(Json) |
| related_task_id | VARCHAR(64) | DEFAULT NULL | 关联异步任务编号 |
| task_success_count | INT | DEFAULT NULL | 异步任务成功数量 |
| task_failure_count | INT | DEFAULT NULL | 异步任务失败数量 |
| task_completed_time | DATETIME | DEFAULT NULL | 异步任务完成时间 |
| created_at | DATETIME | NOT NULL | 记录创建时间 |
| updated_at | DATETIME | NOT NULL | 记录更新时间 |

### 索引

| 索引名 | 字段 | 类型 | 说明 |
|--------|------|------|------|
| PRIMARY | id | PRIMARY | 主键索引 |
| idx_operation_time | operation_time DESC | INDEX | 操作时间降序索引 |
| idx_operator_id_time | operator_id, operation_time DESC | INDEX | 操作人+时间索引 |
| idx_operation_type_time | operation_type, operation_time DESC | INDEX | 操作类型+时间索引 |
| idx_target | target_type, target_id | INDEX | 操作对象索引 |
| idx_related_task | related_task_id | INDEX | 关联任务索引 |

### 操作类型枚举

| 操作类型 | 说明 |
|----------|------|
| TAX_CLASS_CREATE | 报税方案创建 |
| TAX_CLASS_UPDATE | 报税方案更新 |
| TAX_CLASS_DELETE | 报税方案删除 |
| PERSON_ADD | 人员新增 |
| PERSON_UPDATE | 人员更新 |
| PERSON_DELETE | 人员删除 |
| PERSON_SUBMIT | 人员报送 |
| TAX_COMPUTE | 算税执行 |
| TAX_DECLARE | 申报提交 |
| TAX_PAYMENT | 缴款执行 |

### 操作结果枚举

| 操作结果 | 说明 |
|----------|------|
| SUCCESS | 成功 |
| FAILURE | 失败 |
| PARTIAL | 部分成功 |

## 2. 数据溯源说明

### 2.1 主数据流程

```
报税方案(wa_tax_class)
    │
    ├── 1:N ──→ 人员信息(bd_psndoc)
    │               │
    │               ├── 1:1 ──→ 境内人员(bd_psndoc_domestic)
    │               └── 1:1 ──→ 境外人员(bd_psndoc_overseas)
    │
    ├── 1:N ──→ 算税记录(wa_tax_compute_record)
    │               │
    │               ├── 1:N ──→ 减免附表(wa_tax_jmsx_detail)
    │               ├── 1:N ──→ 捐赠附表(wa_tax_jz_detail)
    │               └── 1:N ──→ 健康险附表(wa_tax_syjkbx_detail)
    │
    ├── 1:N ──→ 申报汇总(wa_tax_declaration)
    │               │
    │               └── 1:N ──→ 申报明细(wa_tax_declaration_detail)
    │
    ├── 1:N ──→ 专项附加扣除(wa_tax_deduction_special_detail)
    │
    └── 1:N ──→ 6万减除费用(wa_tax_60k_deduction_detail)
```

| wa_tax_compute_record | 算税记录表，业务核心 | ★★★★★ |
| wa_tax_declaration | 申报汇总表，业务核心 | ★★★★★ |
| wa_tax_income_category | 所得分类字典 | ★★★ |
| wa_tax_income_item | 所得项目字典 | ★★★ |
| operation_log | 操作日志表 | ★★★★ |
